# 🌐 CampusGrid
### The Next-Generation Educational Management Ecosystem

CampusGrid is a premium, high-performance web portal designed to bridge the gap between students, educators, and administrators. Built with a focus on **visual excellence** and **security**, it provides a unified platform for school management.

---

## ✨ Key Features

### 👨‍🎓 Role-Aware Dashboards
- **Student Portal**: Track homework, view attendance stats, and receive real-time notices.
- **Teacher Portal**: Manage assignments, finalize class attendance, and send school-wide broadcasts.
- **Principal Hub**: Comprehensive oversight of school infrastructure, personnel, and academic standards.

### 📝 Academic Management
- **Advanced Homework System**: Multi-file uploads, status-based filtering (Pending, Submitted, Overdue), and automated deadline tracking.
- **Interactive Timetables**: Dynamic schedules for both students and staff.

### 🛡️ Security & Reliability
- **RBAC (Role-Based Access Control)**: Strict route protection and unauthorized access prevention.
- **Smart Session Handling**: Secure authentication with refresh token rotation and hydration protection.
- **Media Privacy**: Secure, temporary presigned URLs for all academic resources and profile media.

### 📢 Communication Engine
- **Unified Broadcasts**: Send rich announcements with attachments and targeted audience control.
- **Live Notifications**: Instant visibility for school-wide updates.

---

## 🚀 Technical Architecture

- **Frontend**: [Next.js 15+](https://nextjs.org/) with Turbopack for lightning-fast builds.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/) for a sleek, modern aesthetic.
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/) for robust client-side state.
- **Icons**: [Remix Icon](https://remixicon.com/) for high-quality, consistent iconography.
- **API Layer**: Standardized Axios/Fetch interceptors for secure communication with the CampusGrid Engine.

## 🛠️ Development

### Prerequisites
- Node.js 20+
- npm or pnpm

### Setup
1. **Clone & Install**:
   ```bash
   git clone https://github.com/preetchahal769/campusgrid.git
   cd campusgrid
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
